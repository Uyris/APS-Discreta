# Guia de Contribuição

Obrigado por considerar contribuir para o APS-Discreta!

## Como Contribuir

1. **Fork o repositório**
2. **Crie uma branch para sua feature** (`git checkout -b feature/MinhaFeature`)
3. **Faça commit das suas mudanças** (`git commit -m 'Adiciona MinhaFeature'`)
4. **Push para a branch** (`git push origin feature/MinhaFeature`)
5. **Abra um Pull Request**

## Padrões de Código

- Siga o PEP 8 para código Python
- Adicione docstrings para todas as funções e classes
- Escreva testes para novas funcionalidades
- Mantenha a cobertura de testes acima de 80%

## Executando Testes

```bash
# Executar todos os testes
pytest

# Executar com cobertura
pytest --cov=src tests/
```

## Estrutura do Código

- `src/`: Código fonte principal
- `tests/`: Testes unitários
- `examples/`: Exemplos de uso
- `docs/`: Documentação adicional

## Tópicos para Contribuição

- Algoritmos de ordenação
- Estruturas de dados avançadas
- Algoritmos de grafos
- Problemas de combinatória
- Teoria dos números
- Algoritmos de strings

## Reportando Bugs

Use as Issues do GitHub para reportar bugs. Inclua:
- Descrição clara do problema
- Passos para reproduzir
- Comportamento esperado vs. observado
- Ambiente (versão do Python, SO, etc.)
